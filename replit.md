# Al Ziyara Transport Website

## Project Overview
Fully static bilingual (English/Urdu) transport booking website for Al Ziyara Transport service in Saudi Arabia. Built with pure HTML, CSS, and JavaScript - no frameworks or backend required.

## Business Information
- **Name**: Al Ziyara
- **Type**: Taxi & Transport Service
- **Owner Location**: Pakistan
- **Drivers Location**: Saudi Arabia
- **Target Customers**: Pakistan & India (Umrah, Hajj, Ziyarat travelers)

## Project Architecture
- **Frontend**: Pure HTML5, CSS3, JavaScript (no frameworks)
- **Fonts**: Google Fonts (Poppins for English, Noto Nastaliq Urdu for Urdu)
- **Images**: Local storage for offline compatibility
- **Deployment**: Netlify-ready static site

## Key Features
- Instant language toggle (no page reload)

- Responsive design (mobile, tablet, desktop)
