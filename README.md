# Al Ziyara Transport - Bilingual Website

A fully static bilingual (English/Urdu) transport booking website for Al Ziyara Transport service.

## Features

- **Bilingual Support**: Instant language switching between English and Urdu (RTL)
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Services**: Airport Pickup/Drop, Ziyarat Tours, City-to-City Rides
- **Vehicle Fleet**: 7 vehicles (Camry, Sonata, Elantra, Hiace, Staria, Starex, GMC)
- **Booking System**: WhatsApp and Email integration for booking requests
- **Service Areas**: Makkah, Madina, Jeddah, Taif, Badar

## Technology Stack

- HTML5
- CSS3 (Custom styling, no frameworks)
- Vanilla JavaScript
- Google Fonts (Poppins for English, Noto Nastaliq Urdu for Urdu)

## Project Structure

```
/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # All styling and responsive design
├── js/
│   └── script.js       # Language toggle and form handling
├── images/             # Vehicle images and hero background
├── assets/             # Favicon and other assets
└── _redirects          # Netlify configuration
```

## Deployment

This website is ready to deploy on Netlify:

1. Push to GitHub repository
2. Connect to Netlify
3. Deploy from main branch
4. No build command needed (static site)
